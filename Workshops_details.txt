Workshops : ห้างออนไลน์User type :- Super Admin : ผู้ดูแลระบบห้างออนไลน์,- Admin : เจ้าของร้านค้า,- Sale Retails : หัวหน้าฝ่ายการตลาดของห้างออนไลน์,- Customer : ลูกค้า,Functional Requirement :- Customer :o สามารถค้นหาสินค้าภายในร้านค้าได้o สามารถซื้อสินค้าได้o สามารถดูประวัติการซื้อสินค้าได้- Sale Retails :o สามารถดูยอดขายต่อชั่วโมง/วัน/สัปดาห์/ปี ตามร้านค้าได้o สามารถดูรายชื่อลูกค้าได้และประวัติการซื้อสินค้าได้o สามารถดูยอดขายสินค้ารายตัวได้- Admin :o สามารถเพิ่ม/ลบ/แก้ไขสินค้าในร้านได้o สามารถกำหนดราคาสินค้าได้o สามารถเลือกสินค้าที่จะนำขึ้นไปขายได้o สามารถเปิดปิดร้านค้าตัวเองได้- Super Admin :o สามารถเปิด-ปิดร้านค้าได้o สามารถกำหนดจำนวนสินค้าที่จะนำขึ้นไปขายต่อร้านได้Non-Functional Requirement :- สามารถคำนวณราคาขายได้อย่างถูกต้อง- สามารถบันทึกข้อมูลการซื้อขายได้อย่างถูกต้อง- สามารถทำสรุปยอดขายได้อย่างถูกต้อง- สามารถเชื่อมต่อกับ Application ภายนอกได้- สามารถรักษาความเป็นส่วนตัวของข้อมูลได้
